# Smart Home Controller (C++)

Console application that simulates a smart home system using bitwise operations.

Each device is represented by a single bit inside one variable.

## Devices
- Light
- Power Socket
- Heating
- Air Conditioner
- Alarm System

## Features
- Turn devices ON/OFF
- Display current device states
- Uses bitwise operators (`|`, `&`, `^`, `~`, `<<`)

## Technologies
- C++
- Bitwise operations
- Console application

## How to run
```bash
g++ main.cpp -o smart_home
./smart_home
```
